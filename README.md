❤️ Heart Disease Analysis

1. Project Description
     This project analyzes heart disease data to identify important factors that contribute to heart disease. The objective of the project is to explore relationships between patient health parameters such as age, cholesterol, blood pressure, and other medical attributes to predict the presence of heart disease.
     The analysis is visualized using interactive dashboards and charts to help understand patterns and insights from the data.

2. Tools and Technologies Used
    a) Python
    b) Flask
    c) HTML
    d) CSS
    e) Tableau
    f) SQL

3. Dataset
   The dataset used in this project contains medical information of patients such as age, gender, cholesterol level, blood pressure, chest pain type, maximum heart rate, and other health indicators that help in analyzing and predicting heart disease.
